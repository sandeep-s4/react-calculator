# React Calculator

A modern, responsive calculator built with React Hooks and styled using Tailwind CSS.
It supports basic arithmetic operations, decimals, percentages, sign toggling, and keyboard-like behavior for a smooth user experience.

# Features

- Basic arithmetic operations (+ - / %)
- Fully responsive (mobile, tablet, desktop)
- Clean UI with animated button interactions

# Tech Stack

- React (functional components + hooks)
- Tailwind CSS for styling
- JavaScript (ES6+)
